# 制作过程

## 后端

前置条件:安装nodejs&npm

安装express & express-generator:

```bash
npm install express express-generator -g
```

创建项目/进入文件夹/安装依赖:

```bash
express backend
cd backend
npm install
```

## 前端

前置条件:安装nodejs&npm

安装create-react-app & yarn

```bash
npm install yarn create-react-app -g
```

*yarn*

yarn是一个nodejs包管理工具，比npm好用（事少）

创建项目/进入文件夹/安装依赖:

```bash
yarn create react-app frontend
cd frontend
yarn install
```
